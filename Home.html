Get started with "Online National Polling System"

The Online National Polling System is an online voting site for national voting purposes.
A user will register by using his Fist Name, Last name, National Identification card and the password. A Candidate can register to the
system by providing his/her information but the Election Commission has to approve his/her profile to publish it in the web page.
The site is easy to use, navigate and the site is updated in real time. Subsequently a voter can vote to any Candidate which the user finds desirable. The candidates can check their progress in real time.
Visitors can check the results from Real-Time visualization page.
